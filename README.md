## рҹҡҖ SETUP QUEST вҖ” Step by Step

### вҡ”пёҸ Step 1 вҖ” Clone the Repo

```bash
git clone https://github.com/your-username/fresh-mart-store.git
cd fresh-mart-store
```

### рҹ§Ә Step 2 вҖ” Install Potions (Dependencies)

```bash
npm install
```

### рҹ”‘ Step 3 вҖ” Fill the Secret Scroll (.env)

```env
MONGO_URI=your_mongodb_connection_string
TELEGRAM_BOT_TOKEN=your_telegram_bot_token
TELEGRAM_CHAT_ID=your_telegram_chat_id
ADMIN_PASS=your_admin_password
PORT=3000
```

| рҹ”® Variable | рҹ“Қ Where to Get It |
|:-----------|:------------------|
| `MONGO_URI` | [MongoDB Atlas](https://www.mongodb.com/atlas) вҖ” Free cloud DB |
| `TELEGRAM_BOT_TOKEN` | [@BotFather](https://t.me/botfather) on Telegram |
| `TELEGRAM_CHAT_ID` | Your Telegram user/group ID |
| `ADMIN_PASS` | Your own secret password |

> рҹӣЎпёҸ **SHIELD RULE:** Never commit `.env` to GitHub! Already in `.gitignore` вң…

### рҹҸғ Step 4 вҖ” Run the Game!

```bash
npm start
```

рҹҺү Open **[http://localhost:3000](http://localhost:3000)** вҖ” You're in!

```
  PUBLIC ROUTES (Anyone can use)
  в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җ
  GET   /api/menu              вҶ’  рҹ“Ӣ Fetch all menu items
  GET   /api/status            вҶ’  рҹҹў Shop open/closed?
  POST  /api/order             вҶ’  рҹ“Ұ Place a new order
  POST  /api/verify-coupon     вҶ’  рҹҺҹпёҸ  Check coupon validity
  POST  /api/order-status      вҶ’  рҹ”Қ Track your order

  ADMIN ROUTES рҹ”җ (Protected)
  в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җ
  POST  /api/admin/login       вҶ’  рҹ”‘ Authenticate admin
  GET   /api/orders            вҶ’  рҹ“Ҡ View recent orders
  POST  /api/admin/add-item    вҶ’  вһ• Add menu item
  POST  /api/admin/delete-item вҶ’  вқҢ Remove menu item
  GET   /api/admin/revenue     вҶ’  рҹ’° Revenue analytics
```
